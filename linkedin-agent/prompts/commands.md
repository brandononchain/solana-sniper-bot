# Command Reference

Quick commands users can say to trigger specific actions.

---

## Content Commands

### Write a Post
```
"Write a post about [topic]"
"Create a post on [topic]"
"Draft content about [topic]"
```

**Agent behavior:**
1. Check voice profile
2. Check what hooks work best
3. Generate 2-3 options
4. Explain strategy behind each

---

### Generate Hooks
```
"Give me hooks for [topic]"
"Generate 10 hooks about [topic]"
"Hook ideas for [topic]"
```

**Agent behavior:**
1. Pull hook patterns that work
2. Apply to topic
3. Generate variations
4. Calibrate to voice

---

### Lead Magnet Ideas
```
"Create a lead magnet about [topic]"
"Lead magnet ideas"
"What lead magnet should I create?"
```

**Agent behavior:**
1. Analyze ICP pain points
2. Match to formats
3. Generate 3-5 concepts with structure

---

### Content Calendar
```
"Plan my content for the week"
"What should I post this week?"
"Content ideas for [topic]"
```

**Agent behavior:**
1. Review what's worked
2. Check trending topics
3. Propose 3-5 posts with hooks

## Natural Language Understanding

The agent should understand variations:
- "Write me a post" = "Create a post" = "Draft content"
- "Help me reach" = "Outreach to" = "Message"
- "What's working" = "Analyze performance" = "Review my content"

Don't require exact commands — understand intent.
